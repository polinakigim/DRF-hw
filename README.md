# Домашняя работа по DRF и Docker

---

## Описание проекта:
**Данный проект создан для получения знаний для работы с DRF в также Docker.** 

В нем реализованы функции:
* Создания, обновление, просмотр и удаления уроков (`Lesson`)
* Создание, обновления, просмотр и удаления курсов (`Course`)
* Реализована Подписка, позволяющая отслеживать у пользователя обновления (`Subscription`)

## Установка и использование:
1. Клонируйте репозиторий на свое локальное утсройство
2. Используя .env.sample создайте свой .env файл с вашими параметрами
3. Проверьте, что у вас установлен Docker
4. Запустите контейнеры в корневой папке проекта  через команду команду sudo docker-compose up --build
5. Приложение будет доступно по адресу http://localhost:8000
